===================================
UltaStar Manager - Plugin SDK 1.0.0
===================================

!! This version of the Plugin SDK is meant to be used with UltraStar Manager 1.8.0 !!

Take a look at the sources of the default plugins of UltraStar Manager 1.8.0 to get to know how to write plugins.

There are two kinds of plugins:
- remote image source plugins
- task plugins

All plugins are multilingual.

An example of a very simple task plugin is the "Lyrics Plugin".
